# Postmortem: MathLive 数式フォント 404

> **重大度**: high / 発生・対応: 2026-06-22（pre-launch 自己検証）
## 1. 概要
MathLive 統合でフォント資産の配信 + fontsDirectory 設定が漏れ、本番の数式エディタでフォントが 404。公開直後・実ユーザー前に自己検証で発見。
## 2. 時系列
| 時刻 | イベント | 対応 |
|---|---|---|
| 06-20 | learning-workbook tdd（math-field 配線、フォント配備漏れ） | — |
| 06-22 | 本番コンソールで font 404 を発見 | claim→fix→tdd→deploy |
## 3. 影響範囲
影響ユーザー 0（実ユーザー前）。データ損失なし。ダウンタイムなし。
## 4. 検知の経緯
開発者の本番コンソール確認。
## 6. 直接原因 + 根本原因
fontsDirectory 未設定 + フォント資産未配信。MathLive 配備手順が統合・テスト計画から漏れ、ブラウザ実機でのみ露見する 404 が build/unit green をすり抜けた。
## 7. 学習事項
- 良かった点: 公開直後の自己検証で実ユーザー前に発見。
- 改善点: 外部 UI ライブラリの「実行時アセット配備」を統合時に検証する観点が不足。
