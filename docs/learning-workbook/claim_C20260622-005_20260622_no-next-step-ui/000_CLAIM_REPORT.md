## 1. クレーム原文
```
正解。次のステップへ
と出たが、次のステップへ進むUIがない
```
## 2. 分解結果
### 2.1 期待 (Expected)
正解後、（最終ステップなら）習得として記録され、次へ進む/単元を完了してテックツリーに戻る等の導線がある。習得で次ノードがアンロックされる（concept UC#7, SPEC §6.2）。
### 2.2 現実 (Actual)
正解で「正解。次のステップへ」というメッセージが出るだけ。次へ進むボタンも、習得記録（progress mastered / 次ノード unlock）も無い。masterUnitAndUnlock は実装済みだがどこからも呼ばれていない。習得 endpoint（SPEC §2.2 POST /api/units/:slug/master）も未実装。
### 2.3 発生条件
/learn/:slug で正解した後。常時。
### 2.4 影響範囲
全学習ユーザー / 中核の体験ループ（解く→習得→アンロック）が起点で閉じない。
### 2.6 報告者文脈
正解したら次に進んで学習を続けたい / 単元を習得して次を開きたい。
